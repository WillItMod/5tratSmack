# Security policy

5tratSmack is public BETA software. Keep an encrypted backup of both the main
wallet and the trading recovery file before mining, trading, updating or moving
an installation. Never publish private keys, wallet passphrases, recovery
files, raw wallet databases or unredacted API credentials.

For a suspected vulnerability, do not open a public issue containing exploit
details or secrets. Contact `axebench.app@gmail.com` with a concise description,
the affected version and a safe reproduction.

Hardware-compatibility reports may use the public issue form after reviewing
the generated report. Remove passwords, tokens, serial numbers, public IP
addresses and any other identifying information before submission.
